SONGSOURCE.ME

SongSource.me was a website I worked on in a 5 person group during Professor Downing's Fall 2022 Software Engineering Course. The goal of the website was to aggregate data about artists, albums, and tracks into a single place for people to access. Additionally, the website had the secondary goal of bringing more notice to less popular artists rather than only focusing on popular artists.
The main features of the website I mainly focused on were:
  - Backend API
    - Flask Local Testing
    - Python Scripting
      - 7 API Endpoints and multiple parameters for each Endpoint
    - Postman JSON Validation
    - Backend Python Unitesting using Pytest
  - Database
    - MySQL Database
    - SQLAlchemy Python Interaction w/ MySQL Database
  - Disparate API Scraping and Dev Acount Set-up
    - Spotify using Spotipy Python Package
    - Genius using requests Python Package
    - GoogleImages using requests Custom Search Engine and google-image-requests Python Package
  - AWS and Namecheap Website Hosting
