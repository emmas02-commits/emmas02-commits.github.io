UNTITLED ROBOT GAME

Untitled Robot Game was a project I iterated on over the course of the semester in Dr. Abraham's Game Technology course. We used this project as a tech demo for various game engine features we created plugins for in Godot over the course of the semester.
The main features that we collectively contributed to for this project included:
  - Advanced 3D 3rd Person Player Package
    - Full Camera Controls and Tank Controls
    - Full WASD Movement, Jump, Crouch, Glide, Double-Jump, and Sprint
    - Timer/Level Win Condition/Game Loop
  - Multiplayer Networking over LAN and IP
    - Including Full Functionality and Gameplay Loop for Racing Friends
  - Particle Effects
    - Bomb Class with Realistic Physics and One-Shot Particle Effects
    - Collectible Power-up Class that gave the Player Double-Jump Ability with Electric Particle Effects
    - "DeathBox" Lava Particle Effects. Deathbox kills the player on contact, sending them back to the start of the level
  - Complex Level Design, Lighting, and Scene Structure Following Best Game Design Practices
  - Advanced AI
    - Crawler AI's that track the player in an area and will kill the player on contact
    - Flyer AI's that track the player in an area above the ground and drop the aformentioned bombs on the player in an attempt to knock them off the platforms into the lava
  - GLSL Lava Shaders
    - Simulated a "wavey" surface using Godot's built-in GLSL-like shader scripting language
